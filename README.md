## 功能

-   [x] Element UI
-   [x] 登录
-   [x] Dashboard/首页
-   [x] 表格
-   [x] Tab 选项卡
-   [x] 表单
-   [x] 404 / 403

## 安装步骤

```
npm install         // 安装项目依赖，等待安装完成之后，安装失败可用 cnpm 或 yarn

// 开启服务器，浏览器访问 http://localhost:8080
npm run serve

// 执行构建命令，生成的dist文件夹放在服务器下即可访问
npm run build

//基于环境参数构建 环境配置文件.env.[mode] 
npm run test 测试环境
npm run pre  预发布环境
npm run pro 生产环境

```
